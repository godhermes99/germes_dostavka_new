Політика конфіденційності

Останнє оновлення: 28 березня 2026

1. Загальні положення

Ця Політика конфіденційності описує, як мобільний додаток ЇжGo ("Додаток", "ми", "нас") збирає, використовує та захищає персональні дані користувачів ("Ви", "Користувач").

2. Які дані ми збираємо

  • Номер телефону — для реєстрації та авторизації через SMS
  • Ім'я та адреса доставки — для оформлення замовлень
  • Геолокація — для визначення адреси доставки та відстеження кур'єра
  • Дані замовлень — історія замовлень, обрані страви, суми
  • FCM-токен пристрою — для надсилання push-повідомлень про статус замовлення

3. Як ми використовуємо дані

  • Обробка та доставка замовлень
  • Надсилання повідомлень про статус замовлення
  • Відображення кур'єра на карті
  • Покращення якості сервісу
  • Зв'язок з вами у разі необхідності

4. Зберігання та захист даних

Ваші дані зберігаються на захищених серверах Supabase (хмарна інфраструктура). Ми вживаємо технічних та організаційних заходів для захисту персональних даних від несанкціонованого доступу.

5. Платіжні дані

Ми не зберігаємо дані банківських карток. Оплата здійснюється через захищений платіжний шлюз Monobank. Усі платіжні дані обробляються безпосередньо банком.

6. Передача даних третім особам

Ми не продаємо та не передаємо ваші персональні дані третім особам, окрім:

  • Ресторанів-партнерів — для виконання замовлення (тільки ім'я, адреса, деталі замовлення)
  • Кур'єрів — для доставки (тільки адреса та номер телефону)
  • Правоохоронних органів — за обґрунтованим запитом відповідно до законодавства України

7. Права користувача

Ви маєте право:

  • Отримати інформацію про ваші персональні дані
  • Вимагати виправлення неточних даних
  • Вимагати видалення ваших даних
  • Відкликати згоду на обробку даних

Для реалізації цих прав зверніться до нас через контакти.

8. Файли cookie

Вебсайт ЇжGo може використовувати localStorage для збереження налаштувань. Ми не використовуємо cookie для відстеження.

9. Зміни до Політики

Ми можемо оновлювати цю Політику. Про суттєві зміни ми повідомимо через додаток або на сайті.
